# ⚔️ GitHub Battle
Live URL:https://papaya-pothos-f86428.netlify.app/

A web app to search GitHub users and battle two developers based on their total GitHub stars.

## 🚀 Features

- 🔍 Search any GitHub user profile
- 📊 View top 5 repositories  
- ⚔️ Battle mode - compare two users
- ⭐ Winner determined by total stars
- 💾 Smart caching for faster loading
- 📱 Fully responsive design

## 🎯 How to Use

**Single Search:**
1. Enter GitHub username (e.g., octocat)
2. Click Search
3. View profile and repositories

**Battle Mode:**
1. Click "Battle Mode" button
2. Enter two usernames
3. Click "Start Battle!"
4. Winner has more total stars

## 🛠️ Technologies

- HTML5, CSS3, JavaScript (ES6+)
- GitHub REST API
- LocalStorage API
